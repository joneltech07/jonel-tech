Projects

JTech Task Management System

JTech Task Management System is a task management application designed to help users efficiently manage their tasks and schedules. It offers two categories for organizing schedules: daily and monthly. The system includes four major features: account management, member management, schedule management, and task management. Built using Vue.js 3 Composition API and Laravel API, this project was developed as a practice tool to enhance my skills in Vue.js and Laravel.

JavaBengture

A gamified mobile application using Android Studio and Java to teach students the Java programming language. The app combines interactive lessons, challenges, and storytelling elements to create an immersive learning experience. Additionally, the app highlights Benguet's rich cultural heritage by featuring local artifacts and tourist spots, promoting the regions attractions while educating users.

Taxicle

I developed a tricycle booking app using Java, which is divided into three separate applications tailored for users, drivers, and administrators. The app includes a range of features designed to enhance the user experience and streamline operations. These features include account settings, announcements, booking, advance booking, booking history, rules and regulations, terms and conditions, and a fare matrix. This project allowed me to integrate practical functionalities.
